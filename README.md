### How to run using docker
```
docker-compose build

docker-compose up website
```
Go to http://localhost:4000/


### How to build and run
1. sudo gem install bundler
2. bundle install
3. bundle exec jekyll serve

Go to http://localhost:4000/

#### Updating

### Firebase hosting
```bash
firebase init
```
